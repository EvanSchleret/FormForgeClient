# Security Policy

## Supported versions

At this stage, only the latest published version of `@evanschleret/formforgeclient` is considered supported for security fixes.

## Reporting a vulnerability

Please do not open a public issue for security reports.

Use GitHub private advisories:

- https://github.com/EvanSchleret/FormForgeClient/security/advisories/new

Include as much context as possible:

- affected version
- reproduction steps
- potential impact
- suggested mitigation (if any)

## Response process

- Initial triage target: within 5 business days
- If accepted, a fix is prepared and released as soon as possible
- Credit can be included in release notes unless you request anonymity
